# Claude Entry

本文件是 Claude Code 进入 `ai-ui-agentic` 时的轻量入口。

## 首次进入时先做什么

1. 先阅读 `.ai/AI_CONTEXT.md`
2. 将 `.ai/AI_CONTEXT.md` 视为本仓库唯一共享项目上下文入口
3. 不要在本文件中复制项目知识

## 继续阅读规则

- 再按当前任务需要读取 `AGENTS.md`
- 再按需读取 `knowledge/`、`tools/`、`.ai/current-state.md`、`.ai/decisions.md` 等原始文档
- 不要默认一次性加载全部目录
- 历史 handoff 仅作历史参考，不能自动视为当前需求
- 如果文档之间存在冲突，先报告，不要自行改写事实

## Claude 扩展位

- 如后续需要增加 Claude 专有操作约束，可在本文件补充
- Claude 专有约束不得复制或替代 `.ai/AI_CONTEXT.md` 中的共享项目知识
